# Bot de WhatsApp Profissional

Bot modular em Node.js construído para rodar nativamente no Termux e estruturado para versionamento no GitHub.

## 📥 Instalação no Termux

Abra o Termux e execute os seguintes comandos para preparar o ambiente:

```bash
# Atualizar pacotes do Termux
apt update && apt upgrade -y

# Instalar Node.js e Git
apt install nodejs git -y

# Clonar o repositório (Substitua pela sua URL)
# git clone [https://github.com/seu-usuario/seu-repositorio.git](https://github.com/seu-usuario/seu-repositorio.git)
# cd seu-repositorio

# Instalar dependências
npm install
